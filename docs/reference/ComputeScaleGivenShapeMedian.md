# Compute Weibull scale parameter

Compute the scale parameter for the Weibull distribution with a given
median (`dMedian`) and shape parameter (`dShape`).

## Usage

``` r
ComputeScaleGivenShapeMedian(dShape, dMedian)
```

## Arguments

- dShape:

  The shape parameter of the Weibull distribution.

- dMedian:

  The median of the Weibull distribution.
